# Nykaa Inspired Beauty E-Commerce Website

This project is a **Nykaa-inspired beauty and skincare website** built using **HTML, CSS, and JavaScript** and deployed on **Amazon Linux with Nginx**.

It is a front-end e-commerce style project created to practice **website design, interactivity, and cloud deployment**.

## Features

- Responsive homepage inspired by Nykaa
- Product listing with multiple categories
- Search functionality
- Category-based filtering
- Product sorting options
- Add to cart feature
- Wishlist feature
- Product detail modal
- Hero banner slider
- Sidebar navigation
- Newsletter subscription section
- Clean UI with modern styling

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Amazon Linux**
- **Nginx**
- **AWS EC2**
